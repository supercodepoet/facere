# Definition

We are going to update our TODO Item Detail view to make our tag management process more rich. We need to support the following for managing tags:

  - Add/Remove tags on a TODO item
  - Create/Edit/Delete a tag
  - Search for a tag
  - Add color to a tag (should be able to pick a custom color)

# Requirements

You can find the visual layout and business flows in the `todo-list-item-screens.pen` file. Specifically you can find the flow on the following screens:

  - Tag Editor Open
  - Tag Editor - Ellipis Menu
  - Create New Tag
  - Edit Tag
  - Delete Tag Confirm
