# Definition

We need to create the screens for managing our TODO Lists. We need the following screens:

- Blank slate for TODO List listing where we can create a new list
- TODO List listing screen that lists all our TODO lists with the ability to create a new one
- A screen with the ability to create a new TODO List:
  - List name
  - Icon
  - Color
  - Description
  - Starting Template
    - Blank (default)
    - Project
    - Weekly
    - Shopping
- A screen showing the blank slate of the TODO List where we can create our first items

# Requirements
- The TODO List name should be unique across the user's account
- The Blank list template should be the default selection
- The template is required so you should not be able to not select one
- Icon is not required
- Color is required and will default to the first one in the listing
