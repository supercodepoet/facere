# Definition

We need to flesh out our TODO Item Detail Screen. This is the main focus of a TODO Item and we need to support all defined features for managing the item. We need to support the following features:

- The ability to Add/Edit a Note
- The ability to Add/Edit/Remove items from a Checklist
	- The ability to mark a Checklist item as finished
- The ability to Add/Remove Attachments
- The ability to Add/Edit/Remove a Comment
	- Show the number of Comments
	- List all the Comments
	- The ability to Like a Comment
	- The ability to Reply to a Comment
- The ability to Set/Change the Status of an item. Support the following statuses:
	- To Do (default)
	- In Progress
	- On Hold
	- Done
- The ability to Set/Change the Priority of an item. Support the following priorities:
	- Urgent
	- High
	- Normal (default)
	- Low
	- None
- The ability to set/change the Due Date
- The ability to Add/Remove assignees to an item
- The ability to Add/Remove people to be notified when Done
- The ability to Add/Remove tags on an item
- The ability to mark an item complete
- The ability to delete an item

# Requirements

Use the `TODO Item Detail` screen of the `todo-list-item-screens.pen` file as the source of truth for these features and how they will be used. It will also contain other information that should be stored and shown on the screen.
