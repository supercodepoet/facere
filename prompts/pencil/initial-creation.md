I want to create a modern, web based TODO list application called Facere. I know there are a ton of these type applications already on the web. I want to take a new approach to the UI/UX focusing on being fun and friendly. I want this application to be a joy to use and maek the user want to come back and use it to manage all their TODO lists. This application should be responsive to all types of mobile devices, as well as, computer screens.

We need the following screens:
	* Need a signup or signin screen
	* A listing screen of all TODO lists
		* Need ability to create a new TODO list
		* Edit an existing TODO list
		* Delete am existing TODO list (prompting the user for confirmation)
		* Be able to re-order TODO lists
	* A detail screen of the currently selected TODO list
		* You should be able to create sections within a TODO list with name and icon
		* Individual TODO items can be at the root list level or under a specific section
		* Sections should be able to be reordered
		* TODO items should be able to be reordered in a section
		* TODO items should be able to be moved between sections
		* You need to be able to add, edit, and delete TODO items from the list or section
		* You need to be able to mark a TODO as completed
		* A completed TODO should move to the bottom of the list or its section with an visual indication that it is completed
	* A detail screen of a TODO item
		* You should have a status of the item and its name
		* You need to be able to assign a TODO to one or more people
		* You need to be able to set a Due Date for a TODO
		* You need to be able to select people that should be notified when the TODO is complete
		* You need to be able to add Notes to a TODO with rich text
		* You need to be able to have attachments for a TODO
		* You need to be able to add a checklist to a TODO
		* Users in the system need to be able to add comments on the TODO item

For UI/UX inspiration use:
	* Linear
	* Fizzy
	* Duolingo
	* Shopify
